# Maps FMX - Mapas multiplataforma com Delphi

Para Android, você precisa adicionar alguns arquivos JAR obrigatórios, segue link com o passo a passo:
[https://www.tmssoftware.com/site/androidjs.asp](https://www.tmssoftware.com/site/androidjs.asp)

Você tem que adicionar 2 pacotes e ambos estão na pasta de instalação do TMS FNC Core, são eles: JavaScriptHelper.jar e FNCWebChromeClient.jar

Para mim estavam nas seguintes pastas:

```
C:\Users\Cesar\AppData\Local\TMS Setup\Products\tms.fnc.core\Android Support\JavascriptHelper\JavascriptHelper.jar
```
 
```
C:\Users\Cesar\AppData\Local\TMS Setup\Products\tms.fnc.core\Android Support\WebChromeClient\FNCWebChromeClient.jar
```
