# TTMSFNCTimeZone

### ptBR: <br/>
**TTMSFNCTimeZone** é um componente para recuperar informações de **fuso horário** para uma localização geográfica especificada usando um serviço de API existente

### en: <br/>
**TTMSFNCTimeZone** is a component for retrieving **time zone** information for a specified geographic location using an existing API service.
