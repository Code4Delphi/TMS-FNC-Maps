# GeoTIFF

### ptBR: <br/>
**GeoTIFF** é um formato de imagem raster que armazena dados geográficos (como projeção e coordenadas) no próprio arquivo, sendo amplamente usado em GIS

### en: <br/>
**GeoTIFF** is a raster image format that stores geographic data (such as projection and coordinates) within the file itself, and is widely used in GIS
