# ElementContainers

### ptBR: <br/>
**ElementContainers** permitem adicionar elementos HTML/CSS personalizados sobre o mapa e criar controles interativos usando Ações. Seu conteúdo pode ser atualizado dinamicamente via JavaScript em tempo de execução. Além disso, são independentes do serviço de mapas, mantendo o mesmo visual e funcionamento em qualquer provedor.


### en: <br/>
**ElementContainers** allow you to add custom HTML/CSS elements on top of the map and create interactive controls using Actions. Their content can be dynamically updated via JavaScript at runtime. Furthermore, they are independent of the map service, maintaining the same look and feel across any provider.
