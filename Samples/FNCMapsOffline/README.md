# Maps Offline

### ptBR: <br/>
Nos casos em que não houver conexão com a internet, ainda pode ser útil ter um aplicativo de mapas à mão.
Este recurso já está disponível no componente TMSFNCOpenLayers do TMS FNC Maps.

**Tiles e Servidores locais de tiles**

Um servidor local de tiles é um serviço que armazena e fornece localmente os tiles (pequenas imagens do mapa, geralmente PNG/JPEG) usados por componentes de mapas, em vez de buscá-los diretamente na internet a cada visualização.

**Em termos simples,** mapas digitais são formados por quadradinhos de imagem (tiles), organizados por zoom, X e Y.
Um servidor local de tiles responde às requisições desses quadradinhos a partir do seu próprio computador ou rede.

---

### en: <br/>
In cases where an internet connection is not available it can be useful to still have a mapping application at hand.
This feature is available now for TMSFNCOpenLayers in TMS FNC Maps.

**Tiles and Local Tile Servers**

A local tile server is a service that stores and provides locally the tiles (small map images, usually PNG/JPEG) used by map components, instead of fetching them directly from the internet for each view.

**In simple terms,** digital maps are made up of image squares (tiles), organized by zoom, X, and Y.

A local tile server responds to requests for these squares from its own computer or network.
