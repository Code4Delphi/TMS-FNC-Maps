# TTMSFNCStaticMap and TTMSFNCMapsImage

### ptBR: <br/>
**TTMSFNCStaticMap** é um componente para recuperar uma URL para uma imagem de mapa estática para um determinado localização geográfica<br/>
**TTMSFNCMapsImage** é um componente Image que pode recuperar automaticamente uma imagem a partir de uma URL e exibi-lo no formulário. Ele suporta os formatos BMP, PNG, GIF, ICO, JPEG

### en: <br/>
**TTMSFNCStaticMap** is a component to retrieve a URL for a static map image for a given geographic location <br/>
**TTMSFNCMapsImage** is an Image component that can automatically retrieve an image from a URL and display it in the form. It supports BMP, PNG, GIF, ICO, and JPEG formats <br/>
