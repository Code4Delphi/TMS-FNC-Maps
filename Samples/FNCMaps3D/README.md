# 3D maps

### ptBR: <br/>
É possível utilizar mapas 3D com dois serviços: MapBox ou Here.
Nos exemplos utilizamos o **MapBox**, então certifique-se de ter uma chave de API válida do MapBox. Para configurar mapas 3D com o MapBox, siga os passos [deste link](https://www.tmssoftware.com/site/cloudkey.asp#MapBox).


### en: <br/>
It is possible to use 3D maps with two services: MapBox or Here.
In the examples we use **MapBox**, so make sure you have a valid MapBox API key. To configure 3D maps with MapBox, follow the steps [at this link](https://www.tmssoftware.com/site/cloudkey.asp#MapBox).
