# TTMSFNCPlaces

TTMSFNCPlaces é um componente para obter sugestões de preenchimento automático para uma consulta de pesquisa utilizando um serviço de API REST existente. Opcionalmente, uma localização pode ser incluída para priorizar os resultados com base em uma coordenada geográfica.

[Documentação oficial / Official documentation](https://download.tmssoftware.com/doc/tmsfncmaps/components/ttmsfncmaps/#ttmsfncplaces)
